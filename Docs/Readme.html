The Hypnotize Project - Readme

MySQL Database installation:

mysql -u your_username -p < Hypnotize-stable.sql

Remember to secure the admin directory with for example htaccess.

Add this to httpd.conf if you want to use the custom error pages (Only if you use Apache!)

ErrorDocument 401 "/?error=401"
ErrorDocument 404 "/?error=404"
ErrorDocument 500 "/?error=500"
ErrorDocument 503 "/?error=502"

The config files (Hypnotize/includes/config.php and Hypnotize-admin/includes/config.php):

$hostname - The ip adress or hostname to the MySQL server
$username - The username that will be used to connect to the MySQL server
$password - The password that will be used to connect to the MySQL server
$database - The database that is used

How do I change the default color style?

The color style are inn the css folder..

copy one of the existing styles to default.css

Example:
"cp black.css default.css"
(without the quotes (")) this will change from your current color style to the Black color style.

Development functions

* is only in the admin interface
** is only in the user interface
smilies($string) - Changes characters with a smiley ( example: ";-)" without " will be changed out with a image smiley )
parseurls($string) - Parses a text for urls and makes them clickable
mysqldie($string) - Kills the MySQL link and adds the message you have choosen ($string) and the MySQL error message
chchar($string) - Changes some chars with real html code
msg($string) - Prints out the text in a message box
error($string) * - Prints out the text in a error box
info($string)* - Prints out the text in a info box
srverror($string1,$string2) ** - Prints out a server error message
